## 🎯 Project Overview

This is my **first Python project** built using **Object-Oriented Programming (OOP)**.  
The system allows users to manage student data through a menu-driven interface.

---

## ✨ Features

| Feature | Description |
|--------|------------|
| ➕ Add Student | Add new student (name, roll no, marks) |
| 📋 View Students | Display all student records |
| ✏️ Update Student | Update marks using roll number |
| ❌ Delete Student | Remove student from system |
| 🔁 Menu System | Simple CLI-based interaction |

---

## 🧠 How It Works

User Input  
↓  
Menu Selection  
↓  
Operation (Add / View / Update / Delete)  
↓  
Students List (stores objects)  
↓  
Student Class (OOP handles data)
